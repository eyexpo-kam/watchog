# Eyexpo Service Health Checker

# Installation 

Ensure you have a fairly recent version of NodeJs - https://nodejs.org/en/.

```
npm i
npm run build
```

# Run program 

Ideally under a cronjob.

```
npm run start
```

# Run tests 

```
npm run test
```

with code coverage
```
npm run coverage 
```

# Generate documentation 

```
npm run docs
```

# Run linter

```
npm run lint 
```
